# website-hosting

### Step 1: _Open GitHub and Create a Repository_
- Go to GitHub and log in.
- Click on the New Repository button  

     <img width="351" alt="new1" src="https://github.com/user-attachments/assets/dce8d06b-f9ad-416e-96e0-79bc51531a66" />
     
- Enter a repository name (e.g., temp).
  
    ![Screenshot 2025-04-04 195603](https://github.com/user-attachments/assets/703dbe4b-c5e0-4f13-85f3-a59e6cf6ceba)
 

- Choose Public (required for GitHub Pages).

    ![Screenshot 2025-04-04 201805](https://github.com/user-attachments/assets/250dcdfc-0958-4965-80b6-23be58a288f6)

- Click Create Repository.

<img width="800" alt="repo" src="https://github.com/user-attachments/assets/e6d9ba48-ab3c-4c27-ae4e-5d26f9286f87" />

### Step 2: _Upload Website Files_

- Click on ‘creating a new file’. You will be redirected to a new page

  <img width="752" alt="11" src="https://github.com/user-attachments/assets/adf7a1c8-b091-464e-ad88-fbfb9ca88da9" />

- 	Give your file an appropriate name. (e.g. - index.html)
  
      <img width="752" alt="12" src="https://github.com/user-attachments/assets/7f61060e-8e55-417e-8be3-54d11e94769b" />
      
- 	Write or copy-paste your code in the given text field. It will look something like this

  ![Screenshot 2025-04-04 212710](https://github.com/user-attachments/assets/47b35973-5c40-4576-89e3-b29e8c8ec2ec)

  - Click Commit changes to finalize the upload

   <img width="400" alt="13" src="https://github.com/user-attachments/assets/a02521bf-7b13-4c2f-a35c-195ada52b9c6" />

<br>

   <img width="399" alt="14" src="https://github.com/user-attachments/assets/a342af64-4f6c-4cf4-a2ee-d4dd257a32ce" />

### Step 3: _Enable GitHub Pages_

- 	Navigate to Settings in your repository

   <img width="752" alt="15" src="https://github.com/user-attachments/assets/9d8fade6-5946-4fa9-aa74-f3b6411e3438" />


  - Scroll down to the Pages section.

    <img width="299" alt="16" src="https://github.com/user-attachments/assets/94062a92-0f78-4d20-bbd1-b7a33b63d00c" />

- 	Under Branch, click None, then select main from the dropdown.


<img width="650" alt="17" src="https://github.com/user-attachments/assets/c38a19a2-b6fa-4f6c-94a7-166c31fb6277" />


 - 	Click Save to activate GitHub Pages.


<img width="650" alt="18" src="https://github.com/user-attachments/assets/1393ec13-886f-4878-8810-bd19ba0b968e" />

### Step 4: _Access Your Website_

- After a few moments, your website will be live 
- Navigate to Settings, scroll down to the Pages section and click Visit site.

![Screenshot 2025-04-04 220116](https://github.com/user-attachments/assets/75ee9b1b-d88b-4ed8-a47e-2939d5bdead7)

- Open the link in a browser to verify everything is working.
